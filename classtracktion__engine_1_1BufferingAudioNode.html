Detailed Description

AudioNode that buffers its input at a larger buffer size than the calling AudioNode to reduce the ammount of function calls during a render pass.

Constructor & Destructor Documentation

int  minBufferSize 
)

Creates a BufferingAudioNode for a given input AudioNode.

The bufferSize provided here is just a hint, it assures that the buffer sized used will be at least that big but for speed and to avoid using circular buffers the actual buffer size will always be a multiple of the input size requested.
